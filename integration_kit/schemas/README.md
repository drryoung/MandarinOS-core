# Canonical schemas live at repo root

The JSON schemas for the trace contract are **not** duplicated here. They live in the MandarinOS-core repo root:

- **`../../schemas/`** (from this folder, `integration_kit/schemas/`)
- **`schemas/`** (from the MandarinOS-core repo root)

Do not copy schema files into this folder. See [../README.md](./../README.md) for how to use the integration kit.
