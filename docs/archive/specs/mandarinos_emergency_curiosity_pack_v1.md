
# MandarinOS Emergency Curiosity Pack v1

Purpose:
Provide extremely simple curiosity questions beginners can use when they do not understand something.

These phrases work across all conversation topics.

--------------------------------------------------

CORE BEGINNER QUESTIONS

这是什么？
What is this?

什么意思？
What does it mean?

你主要做什么？
What do you mainly do?

难吗？
Is it difficult?

为什么？
Why?

像什么？
What is it like?

--------------------------------------------------

FOLLOW-UP QUESTIONS

可以再说一次吗？
Can you say it again?

可以慢一点吗？
Can you speak slower?

可以简单说吗？
Can you explain simply?

可以举个例子吗？
Can you give an example?

--------------------------------------------------

Is it good?
