
# MandarinOS Marketing Positioning (Locked Draft v1)

Core Brand Idea
MandarinOS is not just a language app. It is a **Conversation Operating System**.

The OS metaphor signals that MandarinOS provides the underlying system that enables
real Mandarin conversations to function, rather than just delivering lessons or vocabulary.

--------------------------------------------------

Primary Hook

Fluent speakers don’t know every word.
They know how to keep the conversation going.

Most language apps train vocabulary.

MandarinOS trains the skills that keep conversations alive.

From your first lesson you begin speaking using practical conversational tools:
- asking natural follow-up questions
- navigating new topics
- recovering when you don’t understand
- exploring ideas with limited vocabulary

You don’t need perfect Mandarin to start speaking.
You need to understand how conversations work.

MandarinOS teaches the **Operating System behind Mandarin conversation**.

--------------------------------------------------

Category Positioning

Duolingo → lessons
Anki → flashcards
HelloTalk → chat

MandarinOS → conversation system

MandarinOS is the first Mandarin learning system built around **conversation architecture**
instead of vocabulary memorization.

--------------------------------------------------

Tagline Options

MandarinOS — The Conversation Operating System for Mandarin

MandarinOS — Learn Mandarin Conversations

MandarinOS — The Conversation OS

--------------------------------------------------

Core Promise

Start speaking Mandarin from the first lesson — even with limited vocabulary.

MandarinOS focuses on:
- conversational competence
- strategic competence
- curiosity-driven dialogue
- topic navigation
- conversation recovery skills

--------------------------------------------------

Underlying Learning Principle

MandarinOS emphasizes **strategic competence** and **discourse competence** — two
essential components of communicative ability that most language apps ignore.

Instead of only training grammar and vocabulary, MandarinOS teaches learners how to:

- stay in conversations
- ask meaningful follow-up questions
- recover when they don't understand
- explore topics naturally
