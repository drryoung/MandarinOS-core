# MandarinOS Support Packs v1

Purpose: Capture reusable conversational language packs used across all
MandarinOS engines.

These packs provide the linguistic scaffolding that allows conversations
to remain natural, flexible, and beginner‑friendly.

This document consolidates:

-   Emergency phrases
-   Tiny conversation fillers
-   Orientation vocabulary
-   High‑utility adjectives
-   Relationship vocabulary

These packs are reusable across all engines.

------------------------------------------------------------------------

# 1. Emergency Conversation Phrases

These allow the learner to recover when comprehension breaks down.

## P1 (Beginner)

什么？\
shénme\
What?

再说一次\
zài shuō yí cì\
Say it again.

慢一点\
màn yì diǎn\
Slower please.

听不懂\
tīng bù dǒng\
I can't understand.

我不懂\
wǒ bù dǒng\
I don't understand.

## P2 (More capable learner)

你可以再说一遍吗？\
nǐ kěyǐ zài shuō yí biàn ma\
Can you say that again?

你可以说慢一点吗？\
nǐ kěyǐ shuō màn yìdiǎn ma\
Can you speak more slowly?

什么意思？\
shénme yìsi\
What does that mean?

They show listening, interest, or surprise.

哦\

这样啊\

然后呢？\

对，对\

我也觉得\

# 3. Orientation Vocabulary

These help learners understand where a place is relative to familiar
reference points.

在广州附近\

远不远？\
yuǎn bù yuǎn\

怎么去？\
zěnme qù\

# 7. Design Principles

1.  Support packs should use short, spoken Chinese.
2.  Vocabulary should be reusable across engines.
3.  Beginners should be able to use these phrases immediately.
4.  Packs should remain small and memorable.

------------------------------------------------------------------------

# 8. Role in MandarinOS Architecture

Support packs sit underneath the conversation engine system.

Conversation Engines\
↓\
Curiosity Triggers\
↓\
Conversation Fillers\
↓\
Repair Phrases\
↓\
Support Packs

These layers allow the system to maintain natural conversational flow.
