PRODUCT UX PRINCIPLES

1) 10-minute daily default
   Minimal friction
   High compliance
   Clear done state

2) Word-first, frame-driven
   Users feel they are learning to speak, not study

3) Visual reinforcement is always optional
   Enabled by a simple toggle
   Never blocks forward progress

4) User controls learn vs practice
   A single slider or dial
   Makes the app feel responsive

5) Diagnostic is invisible
   Feels like a normal session
   Routes learner automatically
