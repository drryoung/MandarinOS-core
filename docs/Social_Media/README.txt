MandarinOS MARP Files

Files included:
- mandarinos-marp-template.md
  Reusable starter template for future social/video decks.

- mandarinos-first-video.marp.md
  Your first presentation populated from the script we already developed.

How to use in Cursor / VS Code:
1. Install the Marp extension if needed.
2. Open the .md file.
3. Use Marp preview to inspect slides.
4. Export to PDF or PPTX from the Marp commands.
5. For CapCut, you can also export each slide as images if that fits your workflow.

Practical note:
This deck is intentionally minimalist and optimized for:
- talking-head + slide cutaways
- bold emphasis moments
- simple on-brand repetition
